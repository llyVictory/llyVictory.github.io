# 程序员不鸭 🦆

Java 后端工程师，专注 **Java + AI Agent** 方向的工程落地。

## 我写什么

这个知识库记录我在两条技术线交叉处的实践：

- **Java 后端工程**：Spring Boot / Spring Cloud、并发与线程模型、JVM 调优、Redis 缓存设计、消息中间件、分库分表、稳定性与可观测性。
- **AI Agent 开发**：LLM 应用框架（LangChain / LangGraph / Spring AI）、Agent 编排、RAG 检索增强、Function Calling / Tool Use、Prompt 工程、Agent 在企业内的落地与边界。
- **基础设施**：Docker、Kubernetes、CI/CD，把上面这些跑稳。

核心观点：**Agent 不是聊天机器人，是一个能调工具、有状态、可工程化的系统**。Java 这套成熟的工程方法论，正是 Agent 从 Demo 走向生产的必要约束。

## 关于本站

我是**程序员不鸭**，做 Java 后端，最近几年在把 AI Agent 往生产里落。

这个站点是我个人的技术知识库，按主题组织成可查阅的体系，而不是写完就沉到时间流底下的博客。核心立场：

> **Agent 不是聊天机器人，是一个能调工具、有状态、可工程化的系统**。Java 这套成熟的工程方法论，正是 Agent 从 Demo 走向生产必须补上的那一课。

## 核心入口

- **[Java 后端](/java/)**：Java 基础、并发、JVM、Spring，后端工程师的基本功。
- **[中间件](/middleware/)**：Redis、消息队列，缓存与异步的设计与坑。
- **[AI 应用开发](/ai/)**：LLM、Agent、RAG，大模型应用工程化。
- **[基础设施](/infra/)**：Docker、Kubernetes，容器化与编排。

## 写作原则

- 不写水文，只写自己验证过的东西。
- 先讲清楚问题，再讲方案，最后讲为什么是这个方案。
- 代码能跑就贴能跑的，不能跑就说明为什么。
